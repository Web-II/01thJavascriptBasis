Datatypes

undefined

De speciale waarde undefined staat voor een niet-gedefinieerde waarde.

  • Een variabele waaraan nog geen waarde is toegekend bevat de waarde undefined.
  • Een functie zonder return instructie, retourneert undefined.
  • undefined evalueert false in een boolean expressie.
let a;
console.log(a); // undefined
if (a) {
    console.log("a is true");
    //deze code wordt niet uitgevoerd, evalueert false
}
// OF 
if (a !== undefined) {
    console.log("a !== undefined");
    //deze code wordt niet uitgevoerd, evalueert false
}
// OF 
if (a === undefined) {
    console.log("a === undefined");
    //deze code wordt uitgevoerd, evalueert true
}
