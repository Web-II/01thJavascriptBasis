Datatypes

null

  • De waarde null staat voor een lege object pointer
  • typeof null retourneert "object"
  • null evalueert false in een boolean expressie
  • null wordt gebruikt bij declaratie van een variabele die later een object zal bevatten
const x = null; 
console.log(x); // null
console.log(typeof x); // object
console.log(x ? true : false); // false
