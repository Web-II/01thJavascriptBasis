Variabelen

Inleiding

Een variabele is een geheugenlocatie met een unieke naam. We gebruiken variabelen om data bij te houden.

JavaScript is een loosely typed language. Dit wil zeggen dat er geen type declaraties zijn, zoals bijvoorbeeld in Java. Dus we maken geen integers, booleans, ....

Dit betekent helemaal niet dat JavaScript geen rekening houdt met datatypes. JavaScript bepaalt zelf op basis van de inhoud het type van de variabele.

Er zijn een aantal basistypes:

  • Tekst
  • Getallen
  • Boolean

De namen van variabelen moeten aan een aantal voorwaarden voldoen:

  • Hoofdlettergevoelig
  • 1 woord, minimaal 1 karakter lang
  • beginnen met een letter, $ of een underscore
  • Spaties en andere speciale karakters zijn NIET toegelaten

Het is een goede gewoonte om CamelCasing toe te passen: elk nieuw woord begint met een hoofdletter.

Kies bovendien altijd betekenisvolle namen voor variabelen.

Er zijn 3 manieren om variabelen te declareren:

  • let
  • var
  • const
