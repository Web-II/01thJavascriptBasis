let

Er wordt gebruik gemaakt van het sleutelwoord let om een variabele te declareren

Je geeft geen type op bij de declaratie van een variabele. Dit betekent niet dan de variabele geen type heeft. JavaScript bepaalt op basis van de inhoud het type

Er geldt block scope. Een block wordt gedefinieerd door {}

Een aantal voorbeelden:

let getal;  
console.log(typeof getal);  // type is undefined
getal = 200;
console.log(typeof getal);  // type is number
getal = 10.5;
console.log(typeof getal);  // type is number
getal = 'nu ben ik plots een string';
console.log(typeof getal);  // type is string  
let naam = 'John';   // declaratie en initialisatie in 1 keer  
console.log(naam);  
let tekst = 'va' + 'kan' + 'tie' + '!!!';    // concatenatie
console.log(tekst);
let jaar = 2019, maand = 1, dag = 14;
console.log(jaar);
