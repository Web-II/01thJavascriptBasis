Er wordt gebruik gemaakt van het sleutelwoord const om een constante te declareren

const constanteNaam = constante waarde

Je geeft geen type op bij de declaratie van een constante. Het type is afhankelijk van de toegekende waarde en wordt door JavaScript zelf afgeleid

Het toekennen van een nieuwe waarde is niet toegestaan! Muteren is wel mogelijk (object), waarover later meer.

Er geldt block scope. Een block wordt gedefinieerd door {}

Een aantal voorbeelden:

const intrest = 0.15;
const saldo = 2000;
const intrestPerJaar = saldo * intrest;
console.log(intrestPerJaar);
intrestPerJaar = 400;   // fout: Assignment to constant variable

const total;    // fout: Missing initializer in const declaration
