Operatoren

Berekeningsoperatoren: +, -, *, /, %, ++, --, unary -, unary +, ** (macht)

Toewijzingsoperatoren: =, *=, /=, %=, +=, -=, <<=,>>=, >>>=, &=, ^=, |=

Vergelijkingsoperatoren: ==, !=, ===, !==, >, >=, <, <=

Logische operatoren: && (AND), || (OR), ! (NOT)

String operatoren: + en +=

Conditionele operatoren: condition ? ifTrue : ifFalse

Vergelijkingsoperatoren

  • == of ===
  • != of !==
  • <
  • >
  • <=
  • >=

Gelijkheid

  • == of != --> Het type wordt niet gecontroleerd. Impliciete type conversie.
console.log('hello' == 'hello'); // true
console.log(1 == 1); // true 
console.log(1 == '1'); // true 
console.log(1 == true); //true
  • == of != --> Het type wordt wel gecontroleerd. Geen impliciete type conversie. Gebruik zo vaak mogelijk ===
console.log('hello' === 'hello'); // true
console.log(1 == 1); // true 
console.log(1 === '1'); // false 
console.log(1 === true); // false
console.log(1 === 1.000000000000001); // false 
console.log(1 === 1.0000000000000001); // true
